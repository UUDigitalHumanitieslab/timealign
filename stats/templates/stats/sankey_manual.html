Sankey diagrams

Sankey diagrams are a specific type of flow diagram, in which the width of the arrows is shown proportionally to the flow quantity.

In our case, the Sankey visualization allows you to compare tense use between two languages. On the left hand side, the annotated tenses in the first source language in a scenario are displayed, while on the right side, the annotated tenses in the first target language are displayed. The width of the arrows indicate how many contexts follow this specific path from source to target tense, while hovering allows you to see the exact number. Clicking on an arrow allows to drill through to the specific contexts. The arrows are coloured based on our default tense colouring.

You are able to add additional lanes by changing the filters on top of the diagram:

  • The first and third filter allow to change the source and target languages.
  • The second filter allows to add an additional lane for a property of the source language. This is especially of interest in corpora that are annotated for formal structure (currently: {% for corpus in corpora %}{{ corpus.title }}{% endfor %}), as this allows you to compare e.g. use of tense in dialogue or narration.
  • The fourth filter allows to add an additional lane for the target language.
